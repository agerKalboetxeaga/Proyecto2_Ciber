Vulnerable-django pausuak:

1- Ingurune birtuala sortu eta karpetan sartu
	> virtualenv vulnerable-django
	> cd vulnerable-django

2- Karpetan dagoen aplikazioa osorik kopiatu

3- Ingurune birtuala martxan jarri: 
	> cd Scripts
	> activate.bat

4- cd ..

5- betekizunak instalatu
	> pip install -r requirements.txt

6- Martxan ipini: 
	> python manage.py runserver
